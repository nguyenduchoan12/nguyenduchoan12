🤖 AI Search System (Machine Learning)

1. TF-IDF Vectorization

Hệ thống sử dụng TF-IDF để chuyển văn bản thành vector số.

  • TF (Term Frequency): số lần từ xuất hiện trong sản phẩm
  • IDF (Inverse Document Frequency): độ hiếm của từ trong toàn bộ database

→ Từ càng hiếm → càng quan trọng

2. Cosine Similarity

Sau khi vector hóa, hệ thống dùng Cosine Similarity để đo độ giống giữa query và sản phẩm.

Similarity = cos(θ)

  • ≈ 1 → rất giống
  • ≈ 0 → không liên quan

3. Smart Price Filtering (Custom AI Logic)

Hệ thống hiểu query người dùng:

  • "nike 100" → lọc giá ≤ 100
  • "giày rẻ" → giá thấp (< 100)
  • "giày cao cấp" → giá cao (> 150)

→ Kết hợp NLP + Rule-based filtering

4. System Flow

  1. User nhập hoặc nói (voice)
  2. Chuyển thành text
  3. Vector hóa bằng TF-IDF
  4. Tính Cosine Similarity
  5. Lọc theo giá
  6. Sort kết quả
